Tema: Ejercicio 1: Categorías y Productos

Práctica: 1

Nombre del Práctica: Ejercicio 1: Categorías y Productos

Objetivo:

El alumno:

  • Desarrollará el BackEnd en Spring Boot para gestionar las operaciones relacionadas con Alumnos.
  • Desarrollará el FrontEnd en Angular utilizando Bootstrap para gestionar las operaciones relacionadas con Categorías y Productos.
  • Integrará el BackEnd – FrontEnd.

Instrucciones:

Elaborar un Reporte en LaTeX/Word. El Reporte deberá incluir la siguiente estructura:

Concepto Puntos
1. Portada Sin
2. Introducción 2
3. Desarrollo 4
4. Conclusiones (Énfasis en posibles errores en la instalación). 3
5. Referencias Bibliográficas 1

Instrucciones de entrega del producto esperado:

  • Deberá subir su reporte a la plataforma (pdf del reporte). (No hay prorroga).
  • Entregar URL del sitio en donde se encuentra alojado el FrontEnd para su revisión. La revisión se realizará después de la fecha de entrega. Ejemplo: Netlify (puede utilizar cualquier similar).
  • Entregar URL del sitio en donde se encuentra alojado el BackEnd para su revisión. La revisión se realizará después de la fecha de entrega. Ejemplo: Render (puede utilizar cualquier similar).
  • Entregar Código Fuente Completo del BackEnd y del FrontEnd.
    • Subir liga del repositorio GitHub, GitLab, Google Drive, etc., para descargar los proyectos.
