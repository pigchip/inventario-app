Gestión de Producto

  • Para crear un producto, ingrese los datos y asegúrese de que el ID sea 0, luego presione el botón "Crear/Actualizar".
  • Para actualizar un producto, ingrese el ID del producto y presione el botón "Cargar Datos". Después, actualice los datos y presione el botón "Crear/Actualizar".
  • Para eliminar un producto, ingrese el ID del producto y presione el botón "Eliminar".
El ID del producto no puede ser negativo
El nombre del producto es requerido
La descripción del producto es requerida
El precio debe ser un número entero positivo
La existencia debe ser un número entero positivo
La categoría es requerida
