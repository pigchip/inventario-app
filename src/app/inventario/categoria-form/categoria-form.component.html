Gestión de Categoría

  • Para crear una categoría, ingrese los datos y asegúrese de que el ID sea 0, luego presione el botón "Crear/Actualizar".
  • Para actualizar una categoría, ingrese el ID de la categoría y presione el botón "Cargar Datos". Después, actualice los datos y presione el botón "Crear/Actualizar".
  • Para eliminar una categoría, ingrese el ID de la categoría y presione el botón "Eliminar".
El ID de la categoría no puede ser negativo
El nombre de la categoría es requerido
La descripción de la categoría es requerida
